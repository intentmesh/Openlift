# Roadmap & Milestones

*(Timeline is estimated and dependent on community contribution velocity.)*

## Phase 1: Digital Utility Belt (Current Focus)
**Goal:** Build trust with mechanics through free, useful tools.
- [ ] **OpenVibe:** Flutter app for vibration analysis (ISO 18738).
- [ ] **SpecScanner:** AI-assisted manual lookup tool.
- [ ] **Pollsheet:** Digital survey tool for modernization planning.

## Phase 2: The IoT Bridge
**Goal:** Connect legacy elevators to the cloud without replacing the controller.
- [ ] **Hardware:** ESP32-S3 CAN sniffer design.
- [ ] **Firmware:** Read-only CANopen parser.
- [ ] **Cloud:** Firebase anomaly detection pipeline.

## Phase 3: The Open Controller
**Goal:** A full replacement controller for modernization.
- [ ] **Reference Hardware:** Dual-MCU carrier board design (KiCad).
- [ ] **Safety Kernel:** formally verified E-Stop and Interlock logic.
- [ ] **Dispatch:** OR-Tools integration for group control.
